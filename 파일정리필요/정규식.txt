/^[A-Za-z0-9]+(\.)?[A-Za-z0-9]$/

-> [ ]+ : 하나이상반복

? :
a? -> 문자 a가 있어도되고, 없어도됨(0개또는1개)
(abc)? -> 문자열 abc가 있어도되고 없어도됨
\d? -> 숫자가 있을 수도, 없을 수도 있음
ex ) colou?r
-> u?는 u가 있어도되고 없어도됨
-> "color", "colour"

ex)  https?
s? 는 s가 있어도되고 없어도됨
-> "http", "https"

ex) a.?b
. -> 아무문자1개
.? -> 아무문자 0개 또는 1개
-> a와 b사이에 문자가 있거나 없거나
-> "ab", "acb"

? -> 0개 또는 1개(optional)
* -> 0개 이상 반복
+ -> 1개 이상 반복
